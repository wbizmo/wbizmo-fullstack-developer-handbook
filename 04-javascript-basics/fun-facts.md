# JavaScript Basics Fun Facts

## Fun Fact 1

JavaScript was created in about 10 days in 1995.

## Fun Fact 2

JavaScript was originally called Mocha, then LiveScript, before becoming JavaScript.

## Fun Fact 3

JavaScript is not the same as Java.

## Fun Fact 4

The same JavaScript language can run in browsers and on servers through Node.js.

## Fun Fact 5

Many modern desktop applications are built using web technologies including JavaScript.

## Fun Fact 6

JavaScript is one of the most widely used programming languages in the world.
