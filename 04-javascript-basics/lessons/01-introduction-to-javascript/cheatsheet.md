# Lesson 01 Cheatsheet: Introduction To JavaScript

JavaScript:
    Programming language for behavior and interactivity.

HTML:
    Structure.

CSS:
    Style.

JavaScript:
    Behavior.

Add JavaScript:

    <script src="script.js"></script>

Print to console:

    console.log("Hello");

Alert:

    alert("Welcome");

Single-line comment:

    // Comment

Multi-line comment:

    /*
      Comment
    */

Key takeaway:
    JavaScript makes webpages interactive.
