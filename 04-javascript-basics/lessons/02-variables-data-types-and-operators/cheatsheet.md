# Lesson 02 Cheatsheet: Variables, Data Types and Operators

let:
    Reassignable variable.

const:
    Variable that should not be reassigned.

String:

    const name = "Ada";

Number:

    const age = 25;

Boolean:

    const isReady = true;

null:

    const user = null;

undefined:

    let email;

typeof:

    typeof "Hello"

Arithmetic:

    +
    -
    *
    /
    %

Assignment:

    +=
    -=
    *=
    /=

Template literal:

    const message = `Hello, ${name}`;

Key takeaway:
    Variables store data that programs use.
