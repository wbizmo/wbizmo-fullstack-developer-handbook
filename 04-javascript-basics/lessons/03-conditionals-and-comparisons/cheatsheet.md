# Lesson 03 Cheatsheet: Conditionals and Comparisons

if:

    if (condition) {
      // code
    }

else:

    if (condition) {
      // code
    } else {
      // code
    }

else if:

    if (condition) {
      // code
    } else if (anotherCondition) {
      // code
    } else {
      // code
    }

Strict equal:

    ===

Strict not equal:

    !==

Greater than:

    >

Less than:

    <

Greater or equal:

    >=

Less or equal:

    <=

AND:

    &&

OR:

    ||

NOT:

    !

Falsy values:

    false
    0
    ""
    null
    undefined
    NaN

Key takeaway:
    Conditionals let programs make decisions.
