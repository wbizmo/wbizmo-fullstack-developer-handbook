# JavaScript Basics Module Cheatsheet

Print:

    console.log("Hello");

Variable:

    let score = 0;
    const name = "Ada";

Data types:

    string
    number
    boolean
    null
    undefined
    object

typeof:

    typeof "Hello"

Template literal:

    `Hello, ${name}`

if:

    if (condition) {
      // code
    }

else:

    if (condition) {
      // code
    } else {
      // code
    }

Strict equality:

    ===

AND:

    &&

OR:

    ||

NOT:

    !

Array:

    const items = ["A", "B"];

Object:

    const user = {
      name: "Ada",
      age: 25
    };

Loop:

    for (const item of items) {
      console.log(item);
    }

Function:

    function greet(name) {
      return `Hello, ${name}`;
    }

Arrow function:

    const greet = (name) => {
      return `Hello, ${name}`;
    };

Select DOM element:

    document.querySelector("#id");

Event listener:

    button.addEventListener("click", () => {
      console.log("Clicked");
    });

Key takeaway:
    JavaScript gives webpages logic, data, interactivity, and behavior.
