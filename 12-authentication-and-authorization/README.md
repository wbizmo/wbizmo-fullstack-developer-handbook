# Authentication & Authorization

## Overview

Authentication and Authorization are among the most important concepts in backend engineering.

Authentication answers:

    Who are you?

Authorization answers:

    What are you allowed to do?

This module teaches modern authentication systems used in production APIs and applications.

## Why This Module Matters

Almost every application requires:

- User accounts
- Login systems
- Protected routes
- Permissions
- Session management
- API security

Poor authentication design can compromise entire systems.

## Learning Outcomes

By the end of this module you will be able to:

- Explain authentication
- Explain authorization
- Hash passwords securely
- Build login systems
- Use JWTs
- Use refresh tokens
- Implement API keys
- Implement RBAC
- Understand OAuth
- Secure authentication systems
