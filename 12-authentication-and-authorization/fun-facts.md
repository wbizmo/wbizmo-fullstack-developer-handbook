# Authentication & Authorization Fun Facts

1. Authentication and authorization are different concepts.
2. Passwords should never be stored as plain text.
3. JWTs are signed, not automatically encrypted.
4. API keys are common in developer platforms.
5. OAuth allows login without sharing passwords with third-party apps.
6. Many major breaches involve weak authentication practices.
7. 401 and 403 are different errors.
8. Good auth systems combine multiple layers of protection.
