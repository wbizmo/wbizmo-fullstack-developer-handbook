# Databases & SQL Fun Facts

1. SQL was first developed in the 1970s.
2. PostgreSQL is older than many modern programming languages.
3. Most business software relies heavily on databases.
4. Google, Amazon, and Netflix all use large-scale databases.
5. A poorly designed schema can slow an application dramatically.
6. Database optimization is a specialized engineering field.
7. Many backend performance issues originate from bad queries.
8. Understanding SQL makes ORM tools easier to use.
