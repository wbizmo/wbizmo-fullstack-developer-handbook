# Developer Foundations Module Cheatsheet

## Core Concepts

Programming:
    Writing instructions for computers.

Program:
    A collection of instructions.

Software:
    Programs and applications.

Hardware:
    Physical computer components.

## Computer Components

CPU:
    Executes instructions.

RAM:
    Temporary memory.

Storage:
    Long-term file storage.

## Operating Systems

- Windows
- Linux
- macOS
- Android
- iOS

## Important File Concepts

File:
    Stores information.

Folder:
    Organizes files.

Path:
    Location of a file or folder.

Current directory:
    .

Parent directory:
    ..

## Essential Tools

- VS Code
- Git
- GitHub
- Node.js
- Terminal

## Useful Commands

Current location:

    pwd

List files:

    ls

Change directory:

    cd folder

Move up:

    cd ..

Create folder:

    mkdir folder

Create file:

    touch file.txt

Delete file:

    rm file.txt

View file:

    cat file.txt

## Key Rule

Understand the computer before trying to master programming.
