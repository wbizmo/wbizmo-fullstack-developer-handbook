# Lesson 05 Cheatsheet: Terminal, PowerShell and Git Bash

## Current Location

PowerShell:

    Get-Location

Git Bash:

    pwd

## List Files

PowerShell:

    Get-ChildItem

Alias:

    ls

CMD:

    dir

Git Bash:

    ls

## Change Directory

    cd foldername

## Move Up One Directory

    cd ..

## Create Folder

    mkdir foldername

## Create File

PowerShell:

    New-Item notes.txt

Git Bash:

    touch notes.txt

## Delete File

PowerShell:

    Remove-Item notes.txt

Git Bash:

    rm notes.txt

## Delete Folder

PowerShell:

    Remove-Item foldername -Recurse

Git Bash:

    rm -rf foldername

## View File Contents

PowerShell:

    Get-Content file.txt

Git Bash:

    cat file.txt

## Clear Screen

    clear

## Key Takeaway

The terminal is one of the most important tools in modern software development.
