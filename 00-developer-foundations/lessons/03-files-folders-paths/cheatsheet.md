# Lesson 03 Cheatsheet: Files, Folders and Paths

## File

Stores information.

Examples:

- app.js
- README.md
- photo.jpg

## Folder

Organizes files and folders.

Also called:

    Directory

## File Structure Example

    project/
        index.html
        styles.css
        script.js

## Common File Extensions

Web:

    .html
    .css
    .js
    .ts

Documentation:

    .md
    .txt

Images:

    .png
    .jpg
    .svg

Data:

    .json
    .csv

## Path

A location that tells the operating system where a file or folder exists.

## Absolute Path

Starts from root.

Example:

    C:\Users\User\Documents\app.js

## Relative Path

Starts from current location.

Examples:

    ./app.js
    ../README.md

## Special Symbols

Current directory:

    .

Parent directory:

    ..

## Important Rule

Wrong path = file not found

## Key Takeaway

Files store data.

Folders organize files.

Paths locate files.
