# Lesson 04 Cheatsheet: Development Tools and Installation

## Essential Developer Tools

- Browser
- VS Code
- Git
- GitHub
- Node.js
- Terminal

## VS Code

Purpose:

- Write code
- Edit files
- Run terminals
- Install extensions

Website:

https://code.visualstudio.com

## Git

Purpose:

- Track changes
- Version control
- Restore history

Website:

https://git-scm.com

## GitHub

Purpose:

- Host repositories
- Collaboration
- Releases

Website:

https://github.com

## Node.js

Purpose:

- Run JavaScript outside browsers

Website:

https://nodejs.org

## Installation Verification

Node:

    node -v

npm:

    npm -v

Git:

    git --version

## PATH

Helps operating systems locate installed programs.

## Golden Rule

Always download software from official sources.

## Key Takeaway

A developer's productivity depends heavily on their tools.
