# Lesson 02 Cheatsheet: Computers and Operating Systems

## Computer

A computer is an electronic machine that:

- Receives input
- Processes data
- Stores information
- Produces output

Basic model:

    Input → Processing → Storage → Output

## Input

Data sent into a computer.

Examples:

- Keyboard typing
- Mouse click
- Touchscreen tap
- Microphone audio

## Processing

The computer working on data.

Main component:

- CPU

## Storage

Where data is kept.

Temporary storage:

- RAM

Permanent storage:

- SSD
- HDD
- USB drive

## Output

The result produced by a computer.

Examples:

- Screen display
- Sound
- Printed document
- Saved file

## Hardware

Physical parts of a computer.

Examples:

- Keyboard
- Mouse
- Monitor
- CPU
- RAM
- SSD

## Software

Programs and applications that run on hardware.

Examples:

- Windows
- Chrome
- VS Code
- Node.js
- WhatsApp

## CPU

Central Processing Unit.

It executes instructions and performs calculations.

## RAM

Random Access Memory.

Temporary memory used by active programs.

## SSD or HDD

Long-term storage for files, programs, and operating systems.

## Operating System

The main software that manages the computer.

It manages:

- Hardware
- Files
- Applications
- Processes
- Permissions
- User interface

## Common Operating Systems

- Windows
- macOS
- Linux
- Android
- iOS

## Process

A running program.

Examples:

- Running Chrome
- Running VS Code
- Running a Node.js server

## Permissions

Rules that control who can:

- Read files
- Edit files
- Delete files
- Run files

## Environment Variable

A system-level value used by programs.

Examples:

    PORT=4000
    DATABASE_URL=example
    API_KEY=example

## Key Developer Lesson

Developers must understand operating systems because software development depends on:

- Files
- Folders
- Paths
- Terminals
- Commands
- Permissions
- Processes
- Environment variables
