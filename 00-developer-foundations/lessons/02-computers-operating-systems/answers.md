# Lesson 02 Answers: Computers and Operating Systems

## Multiple Choice Answers

### 1. What is a computer?

Correct answer: B. An electronic machine that receives input, processes data, stores information, and produces output

A computer works by receiving input, processing it, storing information, and producing output.

### 2. What does CPU mean?

Correct answer: C. Central Processing Unit

The CPU executes instructions and performs processing tasks.

### 3. What is RAM used for?

Correct answer: B. Temporary memory for active programs

RAM stores data that running programs need temporarily.

### 4. Which one is an example of storage?

Correct answer: A. SSD

An SSD stores files permanently until they are deleted.

### 5. Which one is hardware?

Correct answer: C. Keyboard

A keyboard is a physical input device.

### 6. Which one is software?

Correct answer: D. Node.js

Node.js is software used to run JavaScript outside the browser.

### 7. What is an operating system?

Correct answer: B. The main software that manages the computer

The operating system controls hardware, applications, files, and user interaction.

### 8. Which operating system is commonly used on servers?

Correct answer: A. Linux

Linux is widely used for web servers, cloud servers, backend systems, and DevOps.

### 9. What is a process?

Correct answer: B. A running program

When an application runs, the operating system manages it as a process.

### 10. What are permissions used for?

Correct answer: A. Controlling who can read, write, or execute files

Permissions protect files and system operations.

## True or False Answers

### 11. Hardware means physical computer parts.

Correct answer: True

Hardware includes physical parts such as keyboards, screens, CPU, RAM, and storage devices.

### 12. Software tells hardware what to do.

Correct answer: True

Software contains instructions that run on hardware.

### 13. RAM is usually long-term permanent storage.

Correct answer: False

RAM is temporary memory. Storage devices like SSDs and HDDs keep files long term.

### 14. An operating system manages files and applications.

Correct answer: True

The operating system manages files, applications, hardware, processes, and permissions.

### 15. Developers never need to understand their operating system.

Correct answer: False

Developers frequently work with commands, paths, permissions, processes, and environment variables.

## Sample Short Answers

### 16. Explain input, processing, storage, and output.

Input is data sent into the computer. Processing is what the computer does with the data. Storage is where data is kept. Output is the result produced by the computer.

### 17. What is the difference between RAM and storage?

RAM is temporary memory used by running programs. Storage keeps files permanently until they are deleted.

### 18. Name three operating systems.

Examples:

- Windows
- macOS
- Linux
- Android
- iOS

### 19. Why is Linux important for backend and server-side development?

Linux is important because many servers, cloud platforms, containers, and backend systems run on Linux.

### 20. What is an environment variable?

An environment variable is a value stored by the operating system and used by programs. Examples include API keys, database URLs, and port numbers.

### 24. Explain why file permissions matter.

File permissions matter because they control who can read, edit, delete, or execute files. They help protect the system from mistakes and unauthorized actions.

### 25. Why should a full stack developer understand computers and operating systems?

A full stack developer should understand operating systems because development involves files, folders, terminals, commands, processes, installations, permissions, and environment variables.
