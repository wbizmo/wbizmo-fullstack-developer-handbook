# Lesson 01 Cheatsheet: What Is Programming?

## Core Definition

Programming is the process of writing instructions that tell a computer what to do.

## Program

A program is a collection of instructions that performs a task.

Examples:

- Browser
- Mobile app
- Game
- API
- Operating system

## Software

Software means programs and applications that run on a computer.

Examples:

- Chrome
- WhatsApp
- VS Code
- Spotify
- Windows

## Hardware

Hardware means the physical parts of a computer.

Examples:

- Keyboard
- Mouse
- Monitor
- CPU
- RAM
- Storage drive

## Binary

Computers ultimately work with binary.

Binary uses:

    0
    1

## Programming Languages

Programming languages help humans write instructions for computers.

Examples:

| Language | Common Use |
|---|---|
| JavaScript | Web development |
| TypeScript | Large web applications |
| Python | Automation, AI, data |
| PHP | Web applications |
| Java | Enterprise software |
| Go | Cloud and infrastructure |
| Rust | Systems programming |

## Frontend Developer

Builds what users see and interact with.

Common tools:

- HTML
- CSS
- JavaScript
- React

## Backend Developer

Builds the systems behind the application.

Common tools:

- Node.js
- Databases
- APIs
- Authentication
- Server logic

## Full Stack Developer

Works with both frontend and backend systems.

## Platform Engineer

Works on tools, infrastructure, automation, deployment, and developer workflows.

## Important Reminder

Programming is not mainly about memorizing code.

Programming is about:

- Understanding problems
- Breaking problems into steps
- Writing clear instructions
- Testing results
- Improving solutions

## Simple Formula

Problem → Steps → Code → Test → Improve

## Real-Life Algorithm Example

Making tea:

1. Boil water.
2. Add tea.
3. Pour water.
4. Wait.
5. Serve.

A computer program also follows ordered instructions.

## Key Takeaway

Programming is problem-solving with instructions.
