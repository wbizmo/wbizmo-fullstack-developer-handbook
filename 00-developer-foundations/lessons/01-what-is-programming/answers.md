# Lesson 01 Answers: What Is Programming?

## Multiple Choice Answers

### 1. What is programming?

Correct answer: B. Writing instructions for a computer

Programming means creating instructions that a computer can follow.

### 2. A program is:

Correct answer: B. A collection of instructions

A program contains instructions that tell a computer how to perform a task.

### 3. Computers ultimately work with:

Correct answer: B. Binary

Computers process information using binary values: 0 and 1.

### 4. Which one is a programming language?

Correct answer: C. JavaScript

JavaScript is a programming language commonly used for web development.

### 5. Why do programming languages exist?

Correct answer: B. To help humans write instructions more easily

Programming languages make it easier for people to write software.

### 6. Which one is hardware?

Correct answer: B. Keyboard

Hardware refers to physical computer parts.

### 7. Which one is software?

Correct answer: C. WhatsApp

Software refers to programs and applications.

### 8. A frontend developer mainly works on:

Correct answer: A. User interfaces

Frontend developers build what users see and interact with.

### 9. A backend developer mainly works on:

Correct answer: A. APIs, databases, and server logic

Backend developers build the behind-the-scenes systems that power applications.

### 10. A full stack developer works with:

Correct answer: C. Both frontend and backend

Full stack developers understand both sides of an application.

## True or False Answers

### 11. Programming is mainly about memorizing code.

Correct answer: False

Programming is about problem-solving and understanding concepts.

### 12. Computers follow instructions exactly.

Correct answer: True

Computers do not guess. They execute instructions as written.

### 13. Software is the physical part of a computer.

Correct answer: False

Software is made of programs. Hardware is physical.

### 14. Hardware includes things like keyboards, monitors, and storage drives.

Correct answer: True

These are physical computer components.

### 15. Developers often use documentation.

Correct answer: True

Documentation helps developers understand tools, languages, and APIs.

## Sample Short Answers

### 16. Define programming in your own words.

Programming is the process of writing instructions that tell a computer what to do.

### 17. What is a program?

A program is a set of instructions that performs a task.

### 18. Why do computers need precise instructions?

Computers need precise instructions because they do not understand human intention. They only execute the instructions they are given.

### 19. Name three programming languages.

Examples:

- JavaScript
- Python
- TypeScript
- PHP
- Java

### 20. Explain the difference between a frontend developer and a backend developer.

A frontend developer builds the parts of an application users see and interact with. A backend developer builds the server, database, APIs, and logic that power the application.
