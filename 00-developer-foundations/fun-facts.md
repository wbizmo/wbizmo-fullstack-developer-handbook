# Developer Foundations Fun Facts

## Fun Fact 1

The first computer bug was an actual moth found inside a computer.

## Fun Fact 2

The first hard drives were so large they needed entire rooms.

## Fun Fact 3

Linux powers most of the world's web servers.

## Fun Fact 4

Git was created by Linus Torvalds, the creator of Linux.

## Fun Fact 5

## Fun Fact 6
