# Advanced JavaScript

## Overview

This module moves beyond fundamentals into the concepts used in professional software development.

## Why This Module Matters

Modern frameworks, APIs, and applications rely heavily on advanced JavaScript.

Understanding these concepts separates beginners from professional developers.

## Learning Outcomes

By the end of this module you will be able to:

- Use asynchronous programming
- Understand promises
- Use async/await
- Understand closures
- Understand scope
- Use modules
- Work with APIs
- Write reusable code
