# Advanced JavaScript

This module moves beyond beginner JavaScript into the concepts used in real applications.

You will learn scope, closures, callbacks, array methods, destructuring, modules, asynchronous JavaScript, promises, async/await, error handling, JSON, fetch, and API communication.

## Lessons

1. Scope, Hoisting and Closures
2. Advanced Functions and Callbacks
3. Array Methods
4. Destructuring, Spread and Rest
5. Modules and Imports
6. Asynchronous JavaScript
7. Promises, Async and Await
8. Error Handling and Debugging
9. JSON, Fetch and API Basics
