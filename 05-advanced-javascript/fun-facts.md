# Advanced JavaScript Fun Facts

## Fun Fact 1

Closures are one of the most powerful and misunderstood concepts in JavaScript.

## Fun Fact 2

Many popular JavaScript frameworks rely heavily on functions, closures, modules, and async behavior.

## Fun Fact 3

Promises made JavaScript asynchronous code much cleaner than older callback-heavy patterns.

## Fun Fact 4

async and await are built on top of promises.

## Fun Fact 5

JSON became popular because it is simple, readable, and easy for many languages to parse.

## Fun Fact 6

fetch replaced many older browser request patterns and made API calls much easier.

## Fun Fact 7

Array methods like map, filter, and reduce are heavily used in React applications.

## Fun Fact 8

Modules are one reason modern JavaScript apps can scale across many files and components.
