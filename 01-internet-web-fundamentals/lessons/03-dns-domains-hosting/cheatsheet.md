# Lesson 03 Cheatsheet: DNS, Domains and Hosting

Domain:
    Human-readable website name.

DNS:
    Translates domain names to IP addresses.

Hosting:
    Server space for websites and applications.

A Record:
    Domain to IPv4 address.

AAAA Record:
    Domain to IPv6 address.

CNAME:
    Domain alias to another domain.

MX:
    Email routing.

TXT:
    Verification and security.

DNS Propagation:
    Delay before DNS changes update everywhere.

Key flow:

    User enters domain
    DNS finds server address
    Browser connects to hosting server
