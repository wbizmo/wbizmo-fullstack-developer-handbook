# Lesson 03 Answers: DNS, Domains and Hosting

1. A domain name is a human-readable website address.
2. DNS means Domain Name System.
3. DNS translates domain names into IP addresses.
4. Hosting means placing a website or app on a server so people can access it online.
5. A domain is the name. Hosting is where the website lives.
6. An A record points a domain to an IPv4 address.
7. A CNAME points one domain name to another domain name.
8. MX records relate to email servers.
9. DNS propagation is the time it takes for DNS changes to update globally.
10. Vercel, Netlify, Render, DigitalOcean, AWS, Railway.
