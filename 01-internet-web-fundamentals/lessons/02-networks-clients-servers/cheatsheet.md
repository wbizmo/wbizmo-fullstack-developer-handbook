# Lesson 02 Cheatsheet: Networks, Clients and Servers

Network:
    Connected devices that communicate.

Client:
    Requests resources.

Server:
    Provides resources.

Request:
    Message sent from client to server.

Response:
    Message sent from server to client.

Basic flow:

    Client → Request → Server
    Client ← Response ← Server

Frontend:
    Often the client.

Backend:
    Often the server.

API:
    Server interface for applications.
