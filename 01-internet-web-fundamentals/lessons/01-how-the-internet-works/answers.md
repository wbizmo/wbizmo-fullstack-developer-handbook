## Sample Short Answers

### 16. Explain the internet in your own words.

The internet is a worldwide network of connected networks that allows devices to communicate and exchange data.

### 17. Explain the difference between the internet and the web.

The internet is the global network infrastructure. The web is a service that runs on the internet and allows people to access websites through browsers.

### 18. What does an ISP do?

An ISP connects users, homes, offices, and devices to the wider internet.

### 19. What is a packet?

A packet is a small piece of data sent across a network. Large data is broken into packets so it can travel efficiently and reliably.

### 20. Why do developers need to understand the internet?

Developers need to understand the internet because websites, APIs, hosting, authentication, databases, and deployments often depend on network communication.

## Practical Answers

### 21. Draw a simple request journey from browser to server.

Example:

    Browser → Router → ISP → Internet Routers → Server → Response Back To Browser

### 22. Give three examples of clients.

Examples:

- Browser
- Mobile app
- API testing tool
- Desktop application

### 23. Give three examples of servers or server types.

Examples:

- Web server
- API server
- Database server
- File server
- Email server

### 24. Explain what happens when a browser requests a website.

The browser receives a URL, finds the server address through DNS, connects to the server, sends a request, receives a response, downloads resources, and renders the page.
