# Lesson 04 Cheatsheet: HTTP, HTTPS and Requests

HTTP:
    Protocol for web communication.

HTTPS:
    Secure encrypted HTTP.

Request:
    Client asks server for something.

Response:
    Server replies.

GET:
    Retrieve data.

POST:
    Create or submit data.

PUT:
    Replace data.

PATCH:
    Update part of data.

DELETE:
    Delete data.

Common Status Codes:

    200 OK
    201 Created
    400 Bad Request
    401 Unauthorized
    403 Forbidden
    404 Not Found
    500 Server Error

Header:
    Metadata.

Body:
    Main data payload.

Key takeaway:
    APIs are built on HTTP requests and responses.
