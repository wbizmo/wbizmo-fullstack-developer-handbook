# Lesson 04 Answers: HTTP, HTTPS and Requests

1. HTTP means HyperText Transfer Protocol.
2. HTTPS adds encryption and secure communication.
3. A request is a message sent from client to server.
4. A response is a message sent from server to client.
5. GET usually retrieves data.
6. POST usually creates or submits data.
7. DELETE usually deletes data.
8. 404 means Not Found.
9. 500 means Server Error.
10. HTTPS protects sensitive data from interception.
