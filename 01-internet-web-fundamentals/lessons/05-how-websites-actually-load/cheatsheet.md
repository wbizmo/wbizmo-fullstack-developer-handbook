# Lesson 05 Cheatsheet: How Websites Actually Load

URL:
    Website address entered by user.

DNS:
    Finds server IP address.

Request:
    Browser asks server for resource.

Response:
    Server replies.

HTML:
    Structure.

CSS:
    Style and layout.

JavaScript:
    Behavior and interactivity.

Network Tab:
    Shows browser requests.

Basic flow:

    URL
    DNS
    Connection
    Request
    Response
    HTML
    CSS
    JavaScript
    Render

Key takeaway:
    A webpage is usually many files loaded through many requests.
