# Internet & Web Fundamentals

Before building websites and web applications, it is important to understand how the internet works.

Every website, API, mobile application, and cloud platform relies on networking concepts.

This module explains the journey data takes across the internet and the technologies that make modern web applications possible.

## Learning Outcomes

After completing this module, you should be able to:

- Explain how the internet works
- Understand networks
- Explain the relationship between clients and servers
- Understand domains, DNS, and hosting
- Understand HTTP and HTTPS
- Explain how websites load in a browser

## Lessons

### 01. How The Internet Works

Learn what the internet is, how data travels, and how computers communicate globally.

### 02. Networks, Clients & Servers

Learn how devices communicate and how client-server architecture powers the web.

### 03. DNS, Domains & Hosting

Learn how domain names work and how websites become accessible online.

### 04. HTTP, HTTPS & Requests

Learn how browsers communicate with servers.

### 05. How Websites Actually Load

Learn the complete journey from entering a URL to seeing a webpage.
