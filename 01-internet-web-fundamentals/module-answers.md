# Internet & Web Fundamentals Module Answers

1. The internet is a global network of connected networks.
2. The internet is infrastructure. The web is websites accessed through browsers.
3. A network is a group of connected devices that communicate.
4. A client requests resources.
5. A server provides resources.
6. DNS translates domains into IP addresses.
7. A domain is a human-readable website name.
8. Hosting places a website or app on a server.
9. HTTP means HyperText Transfer Protocol.
10. HTTPS provides encrypted secure communication.
11. A request is a message from client to server.
12. A response is a message from server to client.
13. GET, POST, PUT, PATCH, DELETE.
14. 404 means Not Found.
15. 500 means Server Error.
16. The browser parses HTML and requests CSS, JavaScript, images, and other resources.
17. Large files, slow servers, poor networks, and unoptimized assets can slow websites.
18. The Network tab shows files, requests, status codes, headers, and timing.
19. URL → DNS → server connection → request → response → resources → render.
20. Full stack developers build apps that depend on browsers, servers, APIs, networks, DNS, HTTP, and hosting.
