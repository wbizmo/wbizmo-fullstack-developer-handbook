# Lesson 08 Cheatsheet

Good folders:

    controllers/
    services/
    routes/
    utils/
    config/

Scripts:

    npm run dev

Ignore:

    node_modules
    .env

Config:

    process.env.PORT

Errors:

    try {
    } catch (error) {
    }

Key takeaway:
    Organized Node.js projects are easier to scale.
