# Lesson 05 Cheatsheet

Import path:

    import path from "node:path";

Join:

    path.join()

Resolve:

    path.resolve()

Current directory:

    process.cwd()

Environment variable:

    process.env.PORT

dotenv:

    import "dotenv/config";

.env:

    PORT=3000

Key takeaway:
    Configuration belongs in environment variables, not source code.
