# Lesson 05: GitHub Fundamentals

## Learning Objectives

- Understand GitHub
- Create repositories
- Connect Git repositories
- Push code
- Clone repositories

## What Is GitHub?

GitHub is a cloud platform for Git repositories.

Git stores history.

GitHub stores and shares repositories.

## Create Repository

On GitHub:

- New Repository
- Name repository
- Create

## Connect Local Repository

    git remote add origin REPOSITORY_URL

Verify:

    git remote -v

## Push Repository

First push:

    git push -u origin main

Later:

    git push

## Clone Repository

    git clone REPOSITORY_URL

## Pull Changes

    git pull

## Exercises

1. Create a GitHub repository.
2. Connect local repository.
3. Push code.
4. Clone repository elsewhere.
5. Pull updates.

## Summary

GitHub hosts repositories online.

Push uploads.

Pull downloads.

Clone copies repositories.
